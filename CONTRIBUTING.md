# Contributing to PY-BEMCS

Thank you for considering contributing to PY-BEMCS! This physics engine was built to be highly modular, and community contributions—whether they are new physics modules, faster solvers, or bug fixes—are incredibly welcome.

## The Rule: Branching Strategy
To keep the engine stable for all researchers and students, this repository uses a strict branching model:
* **`main`**: This branch contains the stable, heavily tested release versions. **Do not submit Pull Requests directly to `main`.**
* **`dev`**: This is the active development branch. All new features, experimental solvers, and bug fixes happen here. 

**All Pull Requests must be directed to the `dev` branch.**
